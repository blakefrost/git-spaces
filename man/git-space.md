git-spaces(1) -- Multiple workspaces for git
=================================================

## SYNOPSIS

`git-spaces` <command> [args]

## DESCRIPTION

  Mutiple working directories for a single repository.

## EXAMPLES

  * Initalize a git working directory for spaces:

    $ git spaces init

  * Create a new space:

    $ git spaces create some-feature
